ตัวอย่าง Data Store Description

Id Data store Structure
D1 แฟ้มข้อมูลผู้ใช้ รหัสผู้ใช้ + อีเมล + รหัสผ่าน + ชื่อ-นามสกุล + [ผู้เรียน | ผู้สอน] + (รหัสนักศึกษา) + (รูปโปรไฟล์) + สถานะยืนยันตัวตน
D2 แฟ้มข้อมูลห้องเรียน รหัสห้องเรียน + ชื่อวิชา + รหัสผู้สอน + กลุ่มเรียน + รหัสเข้าร่วมห้องเรียน
D3 แฟ้มข้อมูลสมาชิกห้องเรียน รหัสการเข้าร่วม + รหัสผู้เรียน + รหัสห้องเรียน + วันที่เข้าร่วม
D4 แฟ้มข้อมูลข้อสอบ รหัสข้อสอบ + รหัสห้องเรียน + ชื่อชุดข้อสอบ + (คำอธิบาย) + คะแนนเต็ม + วันเวลาที่เริ่มและสิ้นสุดการสอบ + รูปแบบการสุ่มข้อสอบ
D5 แฟ้มข้อมูลโจทย์และเกณฑ์ รหัสข้อคำถาม + รหัสข้อสอบ + ข้อความโจทย์ + (รูปภาพประกอบ) + คะแนนประจำข้อ + เฉลยแนวทางคำตอบ + {เกณฑ์การประเมินรูบริค}
D6 แฟ้มข้อมูลการส่งข้อสอบ รหัสการส่งข้อสอบ + รหัสข้อสอบ + รหัสผู้เรียน + วันเวลาที่ส่ง + สถานะการตรวจ + คะแนนรวมที่ได้
D7 แฟ้มข้อมูลคำตอบ รหัสคำตอบ + รหัสการส่งข้อสอบ + รหัสข้อคำถาม + ข้อความคำตอบ + (รูปภาพประกอบคำตอบ) + (คะแนนจาก AI) + (ข้อเสนอแนะจาก AI) + (ระดับความมั่นใจของ AI) + (คะแนนจากผู้สอน) + (ข้อเสนอแนะจากผู้สอน)
D8 แฟ้มข้อมูลรีเซ็ตรหัสผ่าน รหัสคำขอ + รหัสผู้ใช้ + รหัสอ้างอิงการรีเซ็ต + วันเวลาที่ขอ + วันเวลาที่หมดอายุ
D9 แฟ้มข้อมูลยืนยันอีเมล รหัสคำขอ + รหัสผู้ใช้ + รหัสอ้างอิงการยืนยัน + วันเวลาที่ขอ + วันเวลาที่หมดอายุ
D10 แฟ้มข้อมูลประกาศข่าวสาร รหัสประกาศ + รหัสห้องเรียน + รหัสผู้สอน + หัวข้อประกาศ + เนื้อหาประกาศ + วันเวลาที่ประกาศ
D11 แฟ้มข้อมูลการอ่านประกาศ รหัสสถานะการอ่าน + รหัสประกาศ + รหัสผู้เรียน + วันเวลาที่เปิดอ่าน
D12 แฟ้มข้อมูลแจ้งเตือน รหัสแจ้งเตือน + รหัสผู้ใช้ + ประเภทการแจ้งเตือน + ข้อความแจ้งเตือน + วันเวลาที่แจ้ง + สถานะการเปิดอ่าน
