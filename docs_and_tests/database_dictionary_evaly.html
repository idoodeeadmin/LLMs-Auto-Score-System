3.6 การออกแบบฐานข้อมูล (Database Design)

ตารางที่ 3.4 users (ข้อมูลผู้ใช้)

Column Type Description Example Constraint
id INTEGER รหัสอ้างอิงข้อมูลผู้ใช้ 1 PK, Auto Increment
email TEXT ที่อยู่อีเมล student@email.com Unique, Not null
password TEXT รหัสผ่านที่เข้ารหัสแล้ว (Hashed) pbkdf2:sha256... Not null
name TEXT ชื่อ-นามสกุล สมชาย ใจดี Not null
role TEXT ระดับสิทธิ์การใช้งาน (unassigned, teacher, student) student Not null
student_id TEXT รหัสนักศึกษา/รหัสประจำตัว 64010123 Null
avatar_url TEXT รูปโปรไฟล์ผู้ใช้ avatar1.png Null
is_verified INTEGER สถานะการยืนยันอีเมล (0=ยัง, 1=ยืนยันแล้ว) 1 Default 0

ตารางที่ 3.5 rooms (ข้อมูลห้องเรียน)

Column Type Description Example Constraint
id INTEGER รหัสอ้างอิงห้องเรียน 1 PK, Auto Increment
name TEXT ชื่อห้องเรียน/รายวิชา วิศวกรรมซอฟต์แวร์ 101 Not null
section TEXT กลุ่มเรียน (Section) Sec 1 Null
class_code TEXT รหัสเข้าร่วมห้องเรียน XY89AB Unique, Not null
owner_id INTEGER รหัสผู้สร้างห้องเรียน (ผู้สอน) 2 FK -> users(id), Null

ตารางที่ 3.6 enrollments (ข้อมูลการเข้าร่วมห้องเรียน)

Column Type Description Example Constraint
id INTEGER รหัสอ้างอิงการเข้าร่วม 1 PK, Auto Increment
user_id INTEGER รหัสผู้ใช้ (นักเรียน) 5 FK -> users(id), Not null, Unique (with room_id)
room_id INTEGER รหัสห้องเรียน 1 FK -> rooms(id), Not null, Unique (with user_id)
joined_at DATETIME วันและเวลาที่เข้าร่วม 2026-02-15 10:30:00 Default CURRENT_TIMESTAMP

ตารางที่ 3.7 exams (ข้อมูลแบบทดสอบ/ข้อสอบ)

Column Type Description Example Constraint
id INTEGER รหัสอ้างอิงข้อสอบ 1 PK, Auto Increment
room_id INTEGER รหัสห้องเรียน 1 FK -> rooms(id), Not null
title TEXT หัวข้อข้อสอบ สอบกลางภาค Not null
description TEXT คำอธิบายข้อสอบ ข้อสอบอัตนัย 5 ข้อ Null
total_score REAL คะแนนเต็มรวม 50.0 Default 0
start_date TEXT วันเวลาที่เริ่มทำข้อสอบได้ 2026-03-01 09:00 Null
end_date TEXT วันเวลาที่หมดเขตทำข้อสอบ 2026-03-01 12:00 Null
created_at DATETIME วันและเวลาที่สร้าง 2026-02-20 14:00 Default CURRENT_TIMESTAMP

ตารางที่ 3.8 questions (ข้อมูลคำถาม)

Column Type Description Example Constraint
id INTEGER รหัสอ้างอิงคำถาม 1 PK, Auto Increment
exam_id INTEGER รหัสข้อสอบ 1 FK -> exams(id), Not null
text TEXT เนื้อหาคำถาม/โจทย์ จงอธิบายวัฏจักร SDLC Not null
score REAL คะแนนเต็มของข้อนี้ 10.0 Default 0
answer_key TEXT แนวคำตอบ/เฉลย ประกอบด้วย Planning, Analysis... Null
rubrics TEXT เกณฑ์การให้คะแนนรูบริค (JSON) [{'criteria': 'ความถูกต้อง', 'score': 5}] Null
order_index INTEGER ลำดับของข้อคำถาม 1 Default 0
image_path TEXT รูปภาพประกอบโจทย์ q1_img.png Null
image_paths TEXT รายการรูปภาพประกอบ (JSON Array) ['img1.png', 'img2.png'] Null

ตารางที่ 3.9 submissions (ข้อมูลการส่งคำตอบ)

Column Type Description Example Constraint
id INTEGER รหัสอ้างอิงการส่งงาน 1 PK, Auto Increment
exam_id INTEGER รหัสข้อสอบ 1 FK -> exams(id), Not null, Unique (with student_id)
student_id INTEGER รหัสนักเรียนที่ส่งคำตอบ 5 FK -> users(id), Not null, Unique (with exam_id)
status TEXT สถานะการส่ง (missing, submitted, graded) graded Default 'missing'
total_score REAL คะแนนรวมที่ได้รับ 45.5 Default 0
submitted_at DATETIME วันเวลาที่กดส่งคำตอบ 2026-03-01 10:45:00 Null
graded_by_ai INTEGER สถานะการประเมินด้วย AI (0=ยัง, 1=ประเมินแล้ว) 1 Default 0

ตารางที่ 3.10 submission_answers (ข้อมูลคำตอบรายข้อ)

Column Type Description Example Constraint
id INTEGER รหัสอ้างอิงคำตอบรายข้อ 1 PK, Auto Increment
submission_id INTEGER รหัสการส่งงานรวม 1 FK -> submissions(id), Not null, Unique (with question_id)
question_id INTEGER รหัสคำถาม 1 FK -> questions(id), Not null, Unique (with submission_id)
answer_text TEXT ข้อความคำตอบที่นักเรียนพิมพ์ SDLC มี 5 ขั้นตอน... Null
ai_score REAL คะแนนที่ AI ประเมินได้ 8.5 Default 0
ai_feedback TEXT คอมเมนต์อธิบายจาก AI ขาดการอธิบายขั้นตอนที่ 3 Null
ai_confidence TEXT ระดับความมั่นใจของ AI (low, medium, high) high Default 'medium'
teacher_score REAL คะแนนที่ผู้สอนปรับแก้ขั้นสุดท้าย 9.0 Null
teacher_comment TEXT คอมเมนต์จากผู้สอน อธิบายได้ดีมาก แต่เพิ่มน้ำขิงหน่อย Null
image_path TEXT รูปภาพประกอบคำตอบ (ถ้ามี) ans_1.png Null
image_paths TEXT รายการรูปภาพประกอบคำตอบ (JSON Array) ['ans_img1.png'] Null

ตารางที่ 3.11 password_resets (ข้อมูลคำขอรีเซ็ตรหัสผ่าน)

Column Type Description Example Constraint
id INTEGER รหัสอ้างอิง 1 PK, Auto Increment
user_id INTEGER รหัสผู้ใช้งาน 3 FK -> users(id), Not null
token TEXT รหัสโทเคนสำหรับรีเซ็ต abc123xyz... Unique, Not null
expires_at DATETIME เวลาหมดอายุของโทเคน 2026-02-16 12:00:00 Not null

Column Type Description Example Constraint

expires_at DATETIME เวลาหมดอายุของโทเคน 2026-02-16 12:00:00 Not null
