3.10 การออกแบบฐานข้อมูล (Database Design)

ตารางที่ 3.4 ตารางผู้ใช้ (users)
Column Type Description Example Constraint
id INTEGER รหัสของผู้ใช้งานระบบ 1 PK, AUTO_INCREMENT
email VARCHAR(255) อีเมลผู้ใช้งานระบบ ying@email.com UNIQUE, Not Null
password VARCHAR(255) รหัสผ่านผู้ใช้งานระบบ $2b$12$... Not Null
name VARCHAR(255) ชื่อผู้ใช้งานระบบ ปลาวาฬใจดี Not Null
role VARCHAR(50) บทบาทผู้ใช้ (เช่น student, teacher) student Not Null
student_id VARCHAR(100) รหัสประจำตัวนักศึกษา (กรณีเป็นผู้เรียน) 64010123 Null
avatar_url TEXT รูปโปรไฟล์ผู้ใช้งานระบบ http://Profileimg.jpg Null
is_verified TINYINT สถานะการยืนยันตัวตนผ่านอีเมล 1 DEFAULT 0
token_version INTEGER เวอร์ชันของโทเคน 0 DEFAULT 0
ตารางที่ 3.5 ตารางห้องเรียน (rooms)
Column Type Description Example Constraint
id INTEGER รหัสห้องเรียน 101 PK, AUTO_INCREMENT
name VARCHAR(255) ชื่อห้องเรียน วิชาวิทยาการคอมพิวเตอร์ Not Null
section VARCHAR(100) กลุ่มเรียน (Section) Sec 1 Null
class_code VARCHAR(50) รหัสสำหรับเข้าร่วมห้องเรียน A1B2C3 UNIQUE, Not Null
owner_id INTEGER รหัสผู้สร้างห้องเรียน (ผู้สอน) 2 FOREIGN KEY (users.id) ON DELETE SET NULL
ตารางที่ 3.6 ตารางการเข้าร่วมห้องเรียน (enrollments)
Column Type Description Example Constraint
id INTEGER รหัสรายการเข้าร่วมห้องเรียน 500 PK, AUTO_INCREMENT
user_id INTEGER รหัสผู้เรียน 1 FOREIGN KEY (users.id) ON DELETE CASCADE
room_id INTEGER รหัสห้องเรียน 101 FOREIGN KEY (rooms.id) ON DELETE CASCADE
joined_at DATETIME วันเวลาที่เข้าร่วมห้องเรียน 2024-05-01 10:00:00 DEFAULT CURRENT_TIMESTAMP
ตารางที่ 3.7 ตารางข้อสอบ (exams)
Column Type Description Example Constraint
id INTEGER รหัสข้อสอบ 50 PK, AUTO_INCREMENT
room_id INTEGER รหัสห้องเรียน 101 FOREIGN KEY (rooms.id) ON DELETE CASCADE
title VARCHAR(255) ชื่อหัวข้อการสอบ สอบกลางภาค Not Null
description TEXT คำอธิบายการสอบ สอบเก็บคะแนนบทที่ 1-3 Null
total_score FLOAT คะแนนรวมของข้อสอบ 30.0 DEFAULT 0
start_date VARCHAR(100) วันเวลาที่เริ่มสอบ 2024-05-15 09:00:00 Null
end_date VARCHAR(100) วันเวลาที่สิ้นสุดการสอบ 2024-05-15 12:00:00 Null
is_randomized TINYINT สถานะการสลับข้อสอบ 1 DEFAULT 0
created_at DATETIME วันเวลาที่สร้างข้อสอบ 2024-05-10 14:30:00 DEFAULT CURRENT_TIMESTAMP
ตารางที่ 3.8 ตารางโจทย์คำถาม (questions)
Column Type Description Example Constraint
id INTEGER รหัสข้อคำถาม 200 PK, AUTO_INCREMENT
exam_id INTEGER รหัสข้อสอบ 50 FOREIGN KEY (exams.id) ON DELETE CASCADE
text TEXT ข้อความโจทย์คำถาม จงอธิบายหลักการทำงานของ AI Not Null
score FLOAT คะแนนของข้อนี้ 5.0 DEFAULT 0
answer_key TEXT แนวคำตอบ (เฉลย) AI คือปัญญาประดิษฐ์... Null
rubrics TEXT เกณฑ์การให้คะแนน (JSON) [{'criteria':'ความถูกต้อง','score':3}] Null
order_index INTEGER ลำดับข้อคำถาม 1 DEFAULT 0
image_path TEXT ลิงก์รูปภาพประกอบ (1 ภาพ) http://img.jpg Null
image_paths TEXT ลิงก์รูปภาพประกอบ (หลายภาพ JSON) ['http://img1.jpg'] Null
ตารางที่ 3.9 ตารางการส่งข้อสอบ (submissions)
Column Type Description Example Constraint
id INTEGER รหัสการส่งข้อสอบ 1000 PK, AUTO_INCREMENT
exam_id INTEGER รหัสข้อสอบ 50 FOREIGN KEY (exams.id) ON DELETE CASCADE
student_id INTEGER รหัสผู้เรียนที่ส่ง 1 FOREIGN KEY (users.id) ON DELETE CASCADE
status VARCHAR(50) สถานะการส่ง submitted DEFAULT 'missing'
total_score FLOAT คะแนนรวมที่ได้ 28.5 DEFAULT 0
submitted_at DATETIME วันเวลาที่ส่งข้อสอบ 2024-05-15 11:45:00 Null
graded_by_ai TINYINT สถานะการตรวจโดย AI 1 DEFAULT 0
ตารางที่ 3.10 ตารางคำตอบข้อสอบ (submission_answers)
Column Type Description Example Constraint
id INTEGER รหัสคำตอบ 5000 PK, AUTO_INCREMENT
submission_id INTEGER รหัสการส่งข้อสอบ 1000 FOREIGN KEY (submissions.id) ON DELETE CASCADE
question_id INTEGER รหัสข้อคำถาม 200 FOREIGN KEY (questions.id) ON DELETE CASCADE
answer_text TEXT ข้อความคำตอบของผู้เรียน ผมคิดว่า AI คือ... Null
ai_score FLOAT คะแนนที่ AI ประเมินให้ 4.5 DEFAULT 0
ai_feedback TEXT ข้อเสนอแนะจาก AI อธิบายได้ดีแต่ขาดประเด็น X Null
ai_confidence VARCHAR(50) ระดับความมั่นใจของ AI high DEFAULT 'medium'
teacher_score FLOAT คะแนนที่ผู้สอนปรับปรุง 5.0 Null
teacher_comment TEXT ความคิดเห็นจากผู้สอน ตอบได้ครอบคลุมมาก Null
image_path TEXT รูปภาพแนบคำตอบ http://ans.jpg Null
image_paths TEXT รูปภาพแนบคำตอบ (หลายภาพ) ['http://ans1.jpg'] Null
quality_metrics TEXT ข้อมูลชี้วัดคุณภาพคำตอบ {'grammar': 0.9} Null
ตารางที่ 3.11 ตารางรีเซ็ตรหัสผ่าน (password_resets)
Column Type Description Example Constraint
id INTEGER รหัสรายการรีเซ็ต 10 PK, AUTO_INCREMENT
user_id INTEGER รหัสผู้ใช้ 1 FOREIGN KEY (users.id) ON DELETE CASCADE
token VARCHAR(255) โทเคนสำหรับรีเซ็ต abc123xyz UNIQUE, Not Null
expires_at DATETIME วันเวลาที่โทเคนหมดอายุ 2024-06-01 12:00:00 Not Null
ตารางที่ 3.12 ตารางยืนยันอีเมล (email_verifications)
Column Type Description Example Constraint
id INTEGER รหัสรายการยืนยัน 15 PK, AUTO_INCREMENT
user_id INTEGER รหัสผู้ใช้ 1 FOREIGN KEY (users.id) ON DELETE CASCADE
token VARCHAR(255) โทเคนสำหรับยืนยัน def456uvw UNIQUE, Not Null
expires_at DATETIME วันเวลาที่โทเคนหมดอายุ 2024-06-01 12:00:00 Not Null
ตารางที่ 3.13 ตารางประกาศข่าวสาร (announcements)
Column Type Description Example Constraint
id INTEGER รหัสประกาศ 30 PK, AUTO_INCREMENT
room_id INTEGER รหัสห้องเรียน 101 FOREIGN KEY (rooms.id) ON DELETE CASCADE
teacher_id INTEGER รหัสผู้สอนผู้ประกาศ 2 FOREIGN KEY (users.id) ON DELETE CASCADE
title VARCHAR(255) หัวข้อประกาศ เลื่อนวันสอบ Not Null
content TEXT เนื้อหาประกาศ ขอเลื่อนสอบไปวันที่... Not Null
created_at DATETIME วันเวลาที่ประกาศ 2024-05-12 09:00:00 DEFAULT CURRENT_TIMESTAMP
ตารางที่ 3.14 ตารางการอ่านประกาศ (announcement_reads)
Column Type Description Example Constraint
id INTEGER รหัสการอ่าน 400 PK, AUTO_INCREMENT
announcement_id INTEGER รหัสประกาศ 30 FOREIGN KEY (announcements.id) ON DELETE CASCADE
student_id INTEGER รหัสผู้เรียนที่อ่าน 1 FOREIGN KEY (users.id) ON DELETE CASCADE
read_at DATETIME วันเวลาที่อ่านประกาศ 2024-05-12 09:15:00 DEFAULT CURRENT_TIMESTAMP
ตารางที่ 3.15 ตารางการแจ้งเตือน (notifications)
Column Type Description Example Constraint
id INTEGER รหัสการแจ้งเตือน 1500 PK, AUTO_INCREMENT
user_id INTEGER รหัสผู้ใช้ที่รับแจ้งเตือน 1 FOREIGN KEY (users.id) ON DELETE CASCADE
type VARCHAR(50) ประเภทการแจ้งเตือน exam_graded Not Null
message VARCHAR(500) ข้อความแจ้งเตือน ข้อสอบของคุณถูกตรวจแล้ว Not Null
link VARCHAR(255) ลิงก์แนบสำหรับการแจ้งเตือน /exam/result/50 Null
data JSON ข้อมูลแนบเพิ่มเติม {'exam_id': 50} Null
is_read TINYINT(1) สถานะการอ่าน (0 หรือ 1) 0 DEFAULT 0
created_at DATETIME วันเวลาที่แจ้งเตือน 2024-05-16 10:00:00 DEFAULT CURRENT_TIMESTAMP
