ข้อมูลสถานะการส่งอีเมล ข้อมูลคำสั่งและพารามิเตอร์สำหรับการทริกเกอร์ระบบเพื่อส่งอีเมลยืนยันตัวตน Process 1.1 สมัครสมาชิก Process 1.3 ยืนยันอีเมล รหัสผู้ใช้งาน+อีเมล+รหัสโทเคนยืนยันตัวตน+วันเวลาที่ส่งคำสั่ง
ข้อมูลการเข้าสู่ระบบ ข้อมูลอีเมลและรหัสผ่านที่ผู้ใช้งานกรอกเพื่อขอตรวจสอบสิทธิ์เข้าสู่ระบบ ผู้เรียน Process 1.2 เข้าสู่ระบบ อีเมล+รหัสผ่าน

Authentication Token ข้อมูลสถานะการตรวจสอบสิทธิ์และโทเคนยืนยันตัวตนที่ส่งกลับระหว่างระบบกับ Firebase Process 1.2 เข้าสู่ระบบ Firebase Auth รหัสโทเคนยืนยันสิทธิ์+ข้อมูลโปรไฟล์ผู้ใช้+รหัสสถานะ
ข้อมูลอีเมลและลิงก์ยืนยันตัวตน ข้อมูลเนื้อหาอีเมลและลิงก์สำหรับส่งให้ผู้ใช้งานยืนยันตัวตนผ่านเซิร์ฟเวอร์เมล์ Process 1.3 ยืนยันอีเมล SMTP Server อีเมลผู้รับ+หัวข้ออีเมล+เนื้อหาอีเมล+ลิงก์ยืนยันตัวตน
ข้อมูลสถานะการส่งอีเมล ข้อมูลสถานะตอบกลับจากเซิร์ฟเวอร์เมล์เพื่อยืนยันว่าส่งอีเมลสำเร็จหรือล้มเหลว SMTP Server Process 1.3 ยืนยันอีเมล รหัสสถานะเซิร์ฟเวอร์เมล์+ข้อความแจ้งผลการส่ง+วันเวลาที่ส่ง
ข้อมูลอีเมลและลิงก์ยืนยันตัวตน รหัสยืนยันตัวตนที่ผู้ใช้กดผ่านลิงก์ในอีเมลเพื่อยืนยันความเป็นเจ้าของบัญชี ผู้เรียน Process 1.3 ยืนยันอีเมล รหัสผู้ใช้งาน+รหัสโทเคนยืนยันอีเมล

ข้อมูลการส่งคำตอบ ข้อมูลตรวจสอบว่ามีนักเรียนส่งข้อสอบแล้วหรือไม่ เพื่อใช้ล็อกห้ามแก้ไขโครงสร้างข้อสอบ D6 แฟ้มข้อมูลการส่งคำตอบ Process 3.1 จัดการโครงสร้างและรายละเอียดข้อสอบ รหัสข้อสอบ+จำนวนผู้ส่งคำตอบ+สถานะการมีผู้ส่งคำตอบแล้ว
ข้อมูลข้อสอบ ข้อมูลรายละเอียดโครงสร้างข้อสอบ เวลาสอบ เกณฑ์การให้คะแนนรูบริค และสถานะการเผยแพร่ ผู้สอน Process 3.1 จัดการโครงสร้างและรายละเอียดข้อสอบ รหัสข้อสอบ+รหัสห้องเรียน+ชื่อข้อสอบ+คำอธิบายข้อสอบ+คะแนนเต็ม+วันเวลาเริ่มสอบ+วันเวลาสิ้นสุดการสอบ+สถานะการสลับข้อสอบ+รายการคำถาม(รหัสคำถาม+ข้อคำถาม+คะแนน+เฉลย+เกณฑ์รูบริค+ลำดับข้อ+รูปภาพประกอบ)+วันที่สร้าง

ข้อมูลไฟล์รูปภาพโจทย์ข้อสอบ ไฟล์รูปภาพคำถามหรือภาพประกอบโจทย์ที่ผู้สอนอัปโหลดเพื่อส่งไปเก็บที่ Cloudinary ผู้สอน Process 3.2 จัดการรูปภาพประกอบโจทย์ ชื่อไฟล์+ประเภทไฟล์+ข้อมูลรูปภาพฐาน64+ขนาดไฟล์+ตำแหน่งจัดเก็บไฟล์

ข้อมูลโจทย์ข้อสอบและเกณฑ์การให้คะแนน ข้อมูลคำถามและแนวคำตอบที่ส่งไปให้ Gemini AI ช่วยวิเคราะห์และสร้างเกณฑ์รูบริคอัตโนมัติ Process 3.3 จัดการเกณฑ์รูบริคด้วย AI และกำหนดเอง Gemini API รหัสข้อสอบ+ชื่อข้อสอบ+คำอธิบายข้อสอบ+รายการคำถามและแนวคำตอบ+พารามิเตอร์คำสั่งปัญญาประดิษฐ์
ข้อมูลเกณฑ์รูบริคอัตโนมัติ โครงสร้างเกณฑ์รูบริคที่ AI สร้างสำเร็จและส่งกลับมาให้ผู้สอนตรวจสอบก่อนกดบันทึกจริง Gemini API Process 3.3 จัดการเกณฑ์รูบริคด้วย AI และกำหนดเอง รหัสการตอบกลับปัญญาประดิษฐ์+รหัสข้อสอบ+โครงสร้างเกณฑ์รูบริคฉบับร่าง+ค่าความมั่นใจปัญญาประดิษฐ์+วันเวลาที่สร้าง

ข้อมูลข้อสอบ ข้อมูลข้อสอบต้นฉบับที่ดึงมาคัดลอกลงคลังส่วนตัว หรือข้อมูลข้อสอบใหม่ที่ถูกสร้างจากการ Import จากคลัง D4 แฟ้มข้อมูลข้อสอบ Process 4.1 เพิ่มข้อสอบเข้าคลังส่วนตัว รหัสข้อสอบ+รหัสห้องเรียน+ชื่อข้อสอบ+คำอธิบายข้อสอบ+คะแนนเต็ม+วันเวลาเริ่มสอบ+วันเวลาสิ้นสุดการสอบ+สถานะการสลับข้อสอบ+รายการคำถาม(รหัสคำถาม+ข้อคำถาม+คะแนน+เฉลย+เกณฑ์รูบริค+ลำดับข้อ+รูปภาพประกอบ)+วันที่สร้าง
Process 4.2 จัดการและเรียกใช้คลังข้อสอบ D4 แฟ้มข้อมูลข้อสอบ
ข้อมูลการจัดการคลังข้อสอบ คำสั่งจากผู้สอนเพื่อบันทึกข้อสอบและเกณฑ์รูบริคเก็บไว้ในคลังข้อสอบส่วนตัว ผู้สอน Process 4.1 เพิ่มข้อสอบเข้าคลังส่วนตัว รหัสข้อสอบ+รหัสผู้สอน+วันเวลาที่ทำรายการ
ข้อมูลคลังข้อสอบ ข้อมูลรายการข้อสอบและเกณฑ์รูบริคในคลังข้อสอบส่วนตัว (Question Bank) ของผู้สอน Process 4.1 เพิ่มข้อสอบเข้าคลังส่วนตัว ผู้สอน รหัสรายการคลังข้อสอบ+รหัสผู้สอน+ข้อคำถาม+คะแนน+เฉลย+เกณฑ์รูบริค+แท็ก+วันที่สร้าง
Process 4.1 เพิ่มข้อสอบเข้าคลังส่วนตัว D5 แฟ้มข้อมูลคลังข้อสอบ
D5 แฟ้มข้อมูลคลังข้อสอบ Process 4.1 เพิ่มข้อสอบเข้าคลังส่วนตัว
ผู้สอน Process 4.2 จัดการและเรียกใช้คลังข้อสอบ
Process 4.2 จัดการและเรียกใช้คลังข้อสอบ ผู้สอน
Process 4.2 จัดการและเรียกใช้คลังข้อสอบ D5 แฟ้มข้อมูลคลังข้อสอบ
D5 แฟ้มข้อมูลคลังข้อสอบ Process 4.2 จัดการและเรียกใช้คลังข้อสอบ
ข้อมูลการดึงข้อสอบจากคลัง คำสั่งจากผู้สอนเพื่อดึงข้อสอบจากคลังส่วนตัวไปสร้างเป็นข้อสอบใหม่ในห้องเรียน ผู้สอน Process 4.2 จัดการและเรียกใช้คลังข้อสอบ รหัสรายการคลังข้อสอบ+รหัสห้องเรียนปลายทาง+รหัสผู้สอน+วันเวลาที่ดึงไปใช้
Process 5.0: จัดการการทำข้อสอบและส่งคำตอบ
ข้อมูลรหัสข้อสอบเพื่อเริ่มสอบ คำสั่งจากผู้เรียนเพื่อขอเริ่มทำข้อสอบและเริ่มนับเวลาถอยหลังการสอบ ผู้เรียน Process 5.1 จัดการการเริ่มทำข้อสอบและจับเวลา รหัสผู้เรียน+รหัสข้อสอบ+รหัสห้องเรียน+วันเวลาที่ขอเริ่มสอบ
ข้อมูลผู้ใช้งาน ข้อมูลสิทธิ์ผู้เรียนที่ดึงจากฐานข้อมูลเพื่อตรวจสอบสถานะก่อนอนุญาตให้เข้าทำข้อสอบ D1 แฟ้มข้อมูลผู้ใช้งาน Process 5.1 จัดการการเริ่มทำข้อสอบและจับเวลา รหัสผู้ใช้งาน+อีเมล+รหัสผ่าน+ชื่อผู้ใช้งาน+บทบาทผู้ใช้+รหัสประจำตัวนักศึกษา+รูปโปรไฟล์+สถานะการยืนยันตัวตน+เวอร์ชันโทเคน
ข้อมูลข้อสอบ ข้อมูลโจทย์คำถามทั้งหมดพร้อมเวลาสอบที่เหลืออยู่ (รวมเวลาที่ได้ขยายเพิ่มถ้ามี) D4 แฟ้มข้อมูลข้อสอบ Process 5.1 จัดการการเริ่มทำข้อสอบและจับเวลา รหัสข้อสอบ+รหัสห้องเรียน+ชื่อข้อสอบ+คำอธิบายข้อสอบ+คะแนนเต็ม+วันเวลาเริ่มสอบ+วันเวลาสิ้นสุดการสอบ+สถานะการสลับข้อสอบ+รายการคำถาม(รหัสคำถาม+ข้อคำถาม+คะแนน+เฉลย+เกณฑ์รูบริค+ลำดับข้อ+รูปภาพประกอบ)+เวลาสอบที่ขยายเพิ่ม+เวลาที่เหลืออยู่
Process 5.1 จัดการการเริ่มทำข้อสอบและจับเวลา ผู้เรียน
ข้อมูลแบบร่างคำตอบ ข้อมูลคำตอบที่ผู้เรียนพิมพ์หรือเขียนมือซึ่งระบบทำการบันทึกอัตโนมัติ (Auto-save) ระหว่างการสอบ ผู้เรียน Process 5.2 บันทึกแบบร่างและอัปโหลดรูปลายมือ รหัสแบบร่าง+รหัสข้อสอบ+รหัสผู้เรียน+ข้อมูลแบบร่างคำตอบ+วันเวลาที่อัปเดตล่าสุด
Process 5.2 บันทึกแบบร่างและอัปโหลดรูปลายมือ D7 แฟ้มข้อมูลแบบร่าง
D7 แฟ้มข้อมูลแบบร่าง Process 5.2 บันทึกแบบร่างและอัปโหลดรูปลายมือ
D7 แฟ้มข้อมูลแบบร่าง Process 5.3 ยืนยันการส่งคำตอบ
ข้อมูลไฟล์รูปภาพคำตอบลายมือเขียน ไฟล์รูปภาพกระดาษคำตอบเขียนมือที่ผู้เรียนอัปโหลดเพื่อส่งไปเก็บที่ Cloudinary ผู้เรียน Process 5.2 บันทึกแบบร่างและอัปโหลดรูปลายมือ ชื่อไฟล์+ประเภทไฟล์+ข้อมูลรูปภาพฐาน64+ขนาดไฟล์+รหัสผู้เรียน+รหัสข้อสอบ+รหัสคำถาม
Process 5.2 บันทึกแบบร่างและอัปโหลดรูปลายมือ Cloudinary API
ข้อมูล URL และสถานะการจัดเก็บรูปภาพ ลิงก์ URL ของรูปภาพคำตอบลายมือที่ได้รับจาก Cloudinary เพื่อนำมาบันทึกลงในแบบร่าง Cloudinary API Process 5.2 บันทึกแบบร่างและอัปโหลดรูปลายมือ รหัสสินทรัพย์คลาวด์+ลิงก์รูปภาพคำตอบลายมือ+รหัสผู้เรียน+รหัสข้อสอบ+รหัสคำถาม+วันเวลาที่อัปโหลด
Process 5.2 บันทึกแบบร่างและอัปโหลดรูปลายมือ D7 แฟ้มข้อมูลแบบร่าง
ข้อมูลการส่งคำตอบ ข้อมูลกระดาษคำตอบฉบับสมบูรณ์ที่ผู้เรียนกดยืนยันส่งเข้าสู่ระบบ พร้อมบันทึกสถานะการส่ง ผู้เรียน Process 5.3 ยืนยันการส่งคำตอบ รหัสการส่งคำตอบ+รหัสข้อสอบ+รหัสผู้เรียน+สถานะการส่ง+คะแนนรวม+วันเวลาที่ส่ง+สถานะการตรวจด้วยAI+รายการคำตอบ(รหัสคำตอบ+รหัสคำถาม+ข้อความคำตอบ+คะแนนAI+คำติชมAI+ระดับความมั่นใจAI+คะแนนผู้สอน+ความคิดเห็นผู้สอน+รูปภาพคำตอบ)

Process 6.0: ประเมินผลคำตอบด้วย AI
ข้อมูลการส่งคำตอบ ข้อมูลคำตอบและ URL รูปภาพลายมือของนักเรียนที่ดึงมารอการตรวจ หรือข้อมูลผลคะแนนและฟีดแบ็กจาก AI ที่นำมาบันทึกอัปเดต D6 แฟ้มข้อมูลการส่งคำตอบ Process 6.1 เตรียมข้อมูลคำตอบและเกณฑ์รูบริค รหัสการส่งคำตอบ+รหัสข้อสอบ+รหัสผู้เรียน+สถานะการส่ง+คะแนนรวม+วันเวลาที่ส่ง+สถานะการตรวจด้วยAI+รายการคำตอบ(รหัสคำตอบ+รหัสคำถาม+ข้อความคำตอบ+คะแนนAI+คำติชมAI+ระดับความมั่นใจAI+คะแนนผู้สอน+ความคิดเห็นผู้สอน+รูปภาพคำตอบ)
Process 6.3 บันทึกคะแนนและข้อเสนอแนะ D6 แฟ้มข้อมูลการส่งคำตอบ

ข้อมูลข้อสอบ ข้อมูลเกณฑ์รูบริคของข้อสอบข้อนั้นที่ดึงมาเพื่อใช้เป็นกรอบในการให้คะแนนของ AI D4 แฟ้มข้อมูลข้อสอบ Process 6.1 เตรียมข้อมูลคำตอบและเกณฑ์รูบริค รหัสข้อสอบ+รายการเกณฑ์การประเมิน(รหัสเกณฑ์+ชื่อเกณฑ์+คำอธิบาย+คะแนนเต็ม+ระดับคะแนน)
ข้อมูลคำตอบของนักเรียน ชุดข้อมูลและคำสั่ง Prompt ฉบับสมบูรณ์ที่ส่งไปยัง Gemini API เพื่อให้ AI ทำการวิเคราะห์และประเมินผล Process 6.1 เตรียมข้อมูลคำตอบและเกณฑ์รูบริค Process 6.2 ประมวลผลและประเมินผ่าน AI คำสั่งระบบปัญญาประดิษฐ์+ข้อความคำตอบผู้เรียน+ลิงก์รูปภาพลายมือผู้เรียน+ข้อคำถามข้อสอบ+รายการเกณฑ์รูบริค+คะแนนเต็ม
Process 6.2 ประมวลผลและประเมินผ่าน AI Gemini API
ผลการประเมินและข้อเสนอแนะอัจฉริยะ ผลคะแนน ค่าความมั่นใจ (Confidence) และคำอธิบายฟีดแบ็กรายจุดที่ AI ตรวจเสร็จและส่งกลับมา Gemini API Process 6.2 ประมวลผลและประเมินผ่าน AI รหัสการส่งคำตอบ+คะแนนรวม+ระดับความมั่นใจปัญญาประดิษฐ์+ข้อเสนอแนะรายจุด+คะแนนรูบริครายข้อ+เวลาที่ใช้ประมวลผล
Process 6.2 ประมวลผลและประเมินผ่าน AI Process 6.3 บันทึกคะแนนและข้อเสนอแนะ
Process 7.0: ตรวจสอบและประกาศผลคะแนน
ข้อมูลข้อสอบ ข้อมูลเฉลยและเกณฑ์รูบริคที่ดึงมาแสดงประกอบให้ผู้สอนใช้เปรียบเทียบในการตรวจทาน D4 แฟ้มข้อมูลข้อสอบ Process 7.1 ตรวจสอบและปรับปรุงคะแนน รหัสข้อสอบ+ชื่อข้อสอบ+แนวคำตอบเฉลย+รายการเกณฑ์รูบริค
ข้อมูลการส่งคำตอบ ข้อมูลรายการคำตอบของนักเรียนพร้อมคะแนน AI ที่ส่งให้ผู้สอนตรวจสอบ การปรับปรุงคะแนนโดยผู้สอน และการประกาศผลสอบให้ผู้เรียน ผู้สอน Process 7.1 ตรวจสอบและปรับปรุงคะแนน รหัสการส่งคำตอบ+รหัสข้อสอบ+รหัสผู้เรียน+สถานะการส่ง+คะแนนรวม+วันเวลาที่ส่ง+สถานะการตรวจด้วยAI+รายการคำตอบ(รหัสคำตอบ+รหัสคำถาม+ข้อความคำตอบ+คะแนนAI+คำติชมAI+ระดับความมั่นใจAI+คะแนนผู้สอน+ความคิดเห็นผู้สอน+รูปภาพคำตอบ)
Process 7.1 ตรวจสอบและปรับปรุงคะแนน ผู้สอน

Process 7.3 อนุมัติและประกาศผลคะแนน ผู้เรียน
ข้อมูลประวัติการแก้ไขคะแนน ข้อมูลบันทึกประวัติการแก้ไขคะแนน (Audit Log) เพื่อความโปร่งใสและสามารถตรวจสอบย้อนหลังได้ Process 7.1 ตรวจสอบและปรับปรุงคะแนน Process 7.2 จัดการประวัติการแก้ไขคะแนน รหัสบันทึกประวัติ+รหัสผู้ใช้งาน+ประเภทการกระทำ+รหัสเอนทิตี+รายละเอียดการแก้ไข+ไอพีแอดเดรส+วันเวลาที่บันทึก
Process 7.2 จัดการประวัติการแก้ไขคะแนน D9 แฟ้มข้อมูลประวัติการใช้งาน

ข้อมูลเงื่อนไขการส่งออกสถิติ คำสั่งจากผู้สอนเพื่อขอส่งออกข้อมูลคะแนนและสถิติของห้องเรียนออกมาเป็นไฟล์เอกสาร ผู้สอน Process 8.3 ส่งออกรายงานและสถิติ รหัสข้อสอบ+รหัสผู้สอน+รูปแบบไฟล์ที่ต้องการส่งออก+วันเวลาที่ส่งคำขอ

ข้อมูลการส่งคำตอบ ข้อมูลผลคะแนนและสถานะการส่งของนักเรียนทุกคนในห้องที่ดึงมาเพื่อเขียนลงไฟล์รายงาน D6 แฟ้มข้อมูลการส่งคำตอบ Process 8.3 ส่งออกรายงานและสถิติ รหัสข้อสอบ+รายการผลสอบ(รหัสผู้เรียน+ชื่อผู้เรียน+คะแนนรวม+สถานะ+ข้อเสนอแนะปัญญาประดิษฐ์+ความคิดเห็นผู้สอน)
ไฟล์รายงานสรุปคะแนน ไฟล์เอกสารสเปรดชีตสรุปผลคะแนนและสถิติการสอบทั้งหมดที่ระบบสร้างสำเร็จและส่งให้ผู้สอนดาวน์โหลด Process 8.3 ส่งออกรายงานและสถิติ ผู้สอน ส่วนหัวรายงาน+รายการผลสอบ(รหัสผู้เรียน+ชื่อผู้เรียน+สถานะการส่ง+คะแนนรวม+ข้อเสนอแนะปัญญาประดิษฐ์+ความคิดเห็นผู้สอน)+วันเวลาที่ส่งออกไฟล์
